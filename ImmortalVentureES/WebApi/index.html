TestController - для проверки работы сервера
/api/test/echo?text=ping

ExpertSystemController - для работы с экспертными системами
/api/expertSystem/getActualSystems - получить список активных систем
/api/expertSystem/getQueriedVariables?esName=Шашлык - получить список запрашиваемых переменных в определенной ЭС
/api/expertSystem/GoConsult - выполнить консультацию. POST-запросом, в теле запроса надо указать систему, цель консультации и значения запрашиваемых переменных. Пример: { FileName: "Шашлык.es", Goal: "Действие", VarValues: [ { Variable: "Вид мяса", Value: "Свинина" }, { Variable: "Время жарки", Value: "Ну минут 10" }, ] }
