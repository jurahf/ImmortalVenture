TestController - для проверки работы сервера
/api/test/echo?text=ping

ExpertSystemController - для работы с экспертными системами
/api/expertSystem/getActualSystems - получить список активных систем
/api/expertSystem/getQueriedVariables?esName=MedicalView - получить список запрашиваемых переменных в определенной ЭС
/api/expertSystem/GoConsult - выполнить консультацию. POST-запросом, в теле запроса надо указать систему, цель консультации и значения запрашиваемых переменных. Пример: { FileName: "MedicalView.es", Goal: "Состояние", VarValues: [ { Variable: "ТемператураЗначением", Value: "от36и5до37и1" }, { Variable: "ПульсЗначением", Value: "от50до85" }, { Variable: "ДавлениеЗначением", Value: "от140до150" } ] }
